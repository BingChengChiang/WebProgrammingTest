# wp1101

同學你好，這份作業將展示出以下功能：

- 可以新增、刪除相簿、也可以更改相簿名稱
- 允許空相簿存在，不過點按時會跳出警告
- 可以新增、刪除圖片。如果你新增的連結無效，那你自己把那張刪掉就好
- 每個按鈕都可以按按看，你可以試著把所有照片和相簿刪光光
- 刪光光之後還是可以繼續按按鈕，他不一定會理你，但絕對不會噴錯
- 顯示出當前照片數
- 按右鍵，複製圖片位置，然後貼到新增相片處，就會顯示你新增的相片。例如：https://i.imgur.com/dG8s1KW.png<br>直接複製網址有可能會Cross-Origin 甚麼鬼的，然後新增的圖片就無法顯示。
